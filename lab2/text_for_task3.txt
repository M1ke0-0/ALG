Python — высокоуровневый язык программирования общего назначения, ориентированный на повышение производительности разработчика и читаемость кода. Синтаксис ядра минималистичен, в то время как стандартная библиотека включает большой объём полезных функций. Python поддерживает несколько парадигм программирования: структурное, объектно-ориентированное, функциональное, аспектно-ориентированное, декларативное и другие. Основные архитектурные черты — динамическая типизация, автоматическое управление памятью, полная интроспекция, механизм обработки исключений, поддержка многопоточных вычислений и удобные высокоуровневые структуры данных. Код в Python организовывается в функции и классы, которые могут объединяться в модули и пакеты. Рекурсия — это метод, при котором функция вызывает саму себя для решения подзадачи меньшего размера, пока не будет достигнуто базовое (терминальное) условие. Мемоизация позволяет кэшировать результаты дорогостоящих вызовов функций и возвращать закэшированный результат при тех же входных данных. Алгоритм — это конечная последовательность чётко определённых, реализуемых инструкций по решению некоторого класса задач. Сложность алгоритма оценивается с помощью нотации O большого, которая описывает верхнюю границу роста времени выполнения или объёма используемой памяти в зависимости от размера входных данных.
